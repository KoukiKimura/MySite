## 配置ルール

- `docs/` 直下には索引用途の `README.md` 以外のファイルを置かない。
- 要件定義書は `docs/要件定義/` に置く。
- サイト共通の設計書は `docs/サイト設計/` に置く。
- ゲーム別の設計書は `docs/ゲーム設計/` に置く。
- レビュー記録は `docs/レビュー/{対象}/` に置く。
- 運用記録は `docs/運用記録/{対象}/` に置く。
- `{対象}` は `サイト` `タイピング` `ブロック崩し` など、文書の対象ごとに切る。
