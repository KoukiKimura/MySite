## 1. ゲーム機能要件

### 1.1 基本仕様
| 項目 | 内容 |
|------|------|
| ジャンル | ローマ字タイピング |
| 描画方式 | HTML5 Canvas |
| 入力方式 | 物理キーボード（モバイルはソフトキーボード検討） |

### 1.2 出題モード選択
| モード | 内容 |
|--------|------|
| 日本語→ローマ字モード | 日本語の単語・文章が表示され、ローマ字で入力 |
| 英語単語モード | 英単語が表示され、そのままタイピング |

### 1.3 難易度設定
| 難易度 | 出題内容 | 制限時間 |
|--------|----------|----------|
| 初級 | 短い単語（2〜4文字程度） | 長め |
| 中級 | 標準的な単語〜短文 | 標準 |
| 上級 | 長文・難しい語彙 | 短め |

### 1.4 ゲームモード
| モード | 内容 |
|--------|------|
| 時間制限モード | 制限時間（60秒等）内にできるだけ多くタイピング |
| 問題数固定モード | 固定問題数（10問・20問等）をこなすタイムを競う |

### 1.5 ゲームフロー
```
1. ゲームメニュー画面（モード・難易度選択）
       ↓
2. カウントダウン（3, 2, 1, Start!）
       ↓
3. ゲームプレイ（出題 → 入力 → 正誤判定 → 次の問題）
       ↓
4. リザルト画面（スコア、正確率、WPM等）
       ↓
5. リトライ / メニューに戻る
```

### 1.6 画面表示要素
- 出題テキスト（日本語表示 + ローマ字ガイド）
- 入力中のローマ字表示（正誤をリアルタイムで色分け）
- 残り時間 / 進捗バー
- 現在のスコア / コンボ数
- ミスタイプ表示

### 1.7 スコアリング
- 正確にタイプした文字数ベース
- ミスタイプによる減点
- コンボボーナス（連続正解）
- WPM（Words Per Minute）算出

### 1.8 出題データ
- JSON形式で出題データを管理
- 難易度別にデータファイルを分離
- 将来的にデータ追加が容易な構造

## 3. DDD ゲームコンテキスト（タイピング）

### 3.1 値オブジェクト (Value Objects)

| 名称 | 説明 |
|------|------|
| `GameMode` | ゲームモード（時間制限 / 問題数固定） |
| `Difficulty` | 難易度（初級 / 中級 / 上級） |
| `QuestionMode` | 出題モード（日本語→ローマ字 / 英語単語） |
| `Score` | スコア値（正タイプ数、ミス数、コンボ等を含む） |
| `RomajiInput` | ローマ字入力の正誤判定結果 |
| `GameConfig` | ゲーム設定（モード・難易度・問題数・制限時間） |

### 3.2 エンティティ (Entities)

| 名称 | 説明 |
|------|------|
| `GameSession` | 1回のゲームプレイセッション（状態・スコア・残り時間等を管理） |
| `Question` | 1問の出題データ（表示テキスト + ローマ字解答パターン） |

### 3.3 ドメインサービス (Domain Services)

| 名称 | 配置層 | 説明 |
|------|--------|------|
| `RomajiConverter` | domain/services | 日本語→ローマ字変換（複数入力パターン対応） |
| `ScoreCalculator` | domain/services | スコア計算ロジック（WPM、コンボ、ランク判定等） |
| `QuestionProvider` | application/services | 出題データの取得・シャッフル（インフラ層に依存するためアプリケーション層に配置） |

### 3.4 ユースケース

| ID | 名称 | 説明 |
|----|------|------|
| UC-1 | ゲーム開始 | 設定からセッション・出題データを初期化する |
| UC-2 | キー入力処理 | キー入力をローマ字照合し正誤・完了・終了を判定する |
| UC-3 | ゲーム終了 | セッションを終了し最終スコアを算出する |
| UC-4 | リトライ | 同一設定で再セッションを開始する |
| UC-5 | メニューに戻る | セッションを破棄し、メニュー画面へ遷移する |

### 3.5 モジュール構成（概要）

```
games/typing/
├── domain/          値オブジェクト・エンティティ・ドメインサービス
├── application/     ユースケース・QuestionProvider・リポジトリI/F
├── infrastructure/  JsonQuestionRepository・出題JSONデータ
└── presentation/    画面・Canvas描画・キーボード入力
```
