## 2. 本番デプロイ手順

```bash
cd web
npm run deploy
```

内部処理:
1. `vite build`
2. `node deploy.mjs`
3. `env/.env.deploy` の FTP 設定を読み込む
4. `dist/` を ConoHa Wing の `DEPLOY_REMOTE_DIR` へアップロードする
